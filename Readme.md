# Financify - Financial Dashboard

## Overview
Financify is a personal financial management web application that helps you track
income, expenses, bills, and savings goals. It supports users in Singapore,
Australia, and Japan.

## Requirements
- A modern web browser (Chrome, Firefox, Edge, or Safari)
- No installation or internet connection required

## How to Run
1. Download or extract the project folder
2. Open `Index.html` in your web browser
3. Create an account to get started

## Getting Started
1. On the login page, click **Create one** to sign up
2. Enter your personal details on the first sign-up page
3. Enter your financial details on the second sign-up page
4. Log in with your email and password to access your dashboard

## Features
- **Dashboard** — Overview of your income, expenses, tax, debt, and savings
- **Expenses Tracker** — Log and categorise daily expenses with charts
- **My Bills** — Track recurring and one-time bills with a calendar view
- **Goals** — Set and track personal financial savings goals
- **Reports** — View financial summaries and download a PDF report
- **Settings** — Update your financial profile and account preferences
- **Help Center** — FAQs and support contact details

## Supported Countries & Currencies
| Country | Currency |
|---|---|
| Singapore | SGD (S$) |
| Australia | AUD (A$) |
| Japan | JPY (¥) |

## Important Notes
- All data is saved locally in your browser — do not clear browser data or your
  account will be lost
- Passwords are securely hashed before storage
- Your session will automatically expire after 30 minutes of inactivity
- The app is best experienced on a desktop browser
